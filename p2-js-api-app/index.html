About eLearn.

E.Learn is a dynamic educational quiz app designed to facilitate learning for users of all ages. Focused on Math and Science, E.Learn offers an interactive experience where users can start by reviewing the quiz mechanics and then proceed to customize their quiz.

Users can select the number of questions (5 or 10), choose the difficulty level (Easy, Medium, or Hard), and set the timer (10, 15, or 20 seconds). With its user-friendly interface and instant feedback, E.Learn makes learning both enjoyable and effective, helping users improve their knowledge in a fun and personalized way.

Quiz Mechanics:

1. Select the number of questions, difficulty level, and timer duration,then click Enter. 2. If a question is answered correctly, the quiz will move to the next question. 3. If a question is answered incorrectly, the correct answer will be highlighted before proceeding to the next question. 4. If the timer expires, the correct answer will be highlighted, and the quiz will proceed to the next question. 5. The final score, reflecting the number of correct answers, will be displayed at the end of the quiz.
